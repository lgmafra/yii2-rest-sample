Yii 2 Rest Sample


REQUIREMENTS
------------

docker and docker-compose


INSTALLATION
------------

### Install 

Clone this repository

Run the container:

~~~
cd yii2-rest-sample

docker-compose up -d
~~~

Download the dependencies.

~~~
composer update
~~~


Test in http://localhost:8081
